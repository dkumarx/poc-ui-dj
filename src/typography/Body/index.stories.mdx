# `<Body />`

This component accepts a `variant` prop, which may be either `body1` or `body2`.
By default, it is equivalent to `<Typography variant="body1">`. It renders a 
styled `<p />` element.

### Usage

```
import { Body } from 'poc-ui-mat-lib';

// or

import Body from 'poc-ui-mat-lib/typography/Body';

// ... inside your component ...

return (
  <>
    <Body>Body 1, default</Body>

    <Body variant="body2">Body 2</Body>
  </>
);
```
