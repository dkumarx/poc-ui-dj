# `<H1 />`

Equivalent to `<Typography variant="h1">`. Renders a styled `<h1 />` element.

### Usage

```
import { H1 } from 'poc-ui-mat-lib';

// or

import H1 from 'poc-ui-mat-lib/typography/H1';

// ... inside your component ...

return <H1>Heading 1</H1>;
```
