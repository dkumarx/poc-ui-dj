# `<H2 />`

Equivalent to `<Typography variant="h2">`. Renders a styled `<h2 />` element.

### Usage

```
import { H2 } from 'poc-ui-mat-lib';

// or

import H2 from 'poc-ui-mat-lib/typography/H2';

// ... inside your component ...

return <H2>Heading 2</H2>;
```
